# Resting Bradoc

A restful resource for Bradoc.

## Getting Started
Install the module with

`git clone resting-bradoc`

`cd resting-bradoc`

`node app.js`

Generators and Validators for CPF and CNPJ are available.

### CPF Restful Resource

`/cpf`

```javascript
{
  "cpf": "735.896.679-15"
}
```

### CNPJ Restful Resource

`/cnpj`

```javascript
{
  "cnpj": "75.936.231/7185-67"
} 
```
